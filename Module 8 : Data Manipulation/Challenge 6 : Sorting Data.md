## What I Learnt
Files (or output lines of commands) aren't always in the order you need them! The sort command helps you organize data.
It reads lines from input (or files) and outputs them in sorted order.
By default, sort orders lines alphabetically. Arguments can change this:

-r: reverse order (Z to A)
-n: numeric sort (for numbers)
-u: unique lines only (remove duplicates)
-R: random order
